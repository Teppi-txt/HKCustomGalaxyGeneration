# HKCustomGalaxyGeneration
A program to generate custom Hollow Knight bingo boards for the galaxy gamemode.

Galaxy is a 4 player bingo gamemode where each player has a "line", and must complete the objectives on their line in order and without marking any other goals. It is a commonly played gamemode in the Hollow Knight community.
